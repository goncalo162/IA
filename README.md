# IA-Projeto


## Dependências

**Textual** Pacote de python para criação de interfaces no terminal

Instalar (Linux/macOS):
    Para a instalação do *Textual* é necessário criar um ambiente virtual python e instalar o pacote através do pip 
    
```
python -m venv venv //Criar um novo ambiente virtual de nome venv
source venv/bin/activate //Ativar o ambiente virtual
python -m pip install textual textual-dev //Instalar os pacotes textual e textual-dev no venv
```
